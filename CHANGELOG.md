# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Initial release of Timeline Fishbone Generator
- Smart layout engine with automatic spacing adjustment
- Comprehensive configuration system (CLI, YAML, JSON, API)
- CSV and JSON data input support
- LaTeX TikZ code generation for fishbone timeline diagrams
- Data validation with detailed error messages
- Command-line interface with extensive options
- Python API for programmatic usage
- Sample data generation
- Complete test suite with >85% coverage
- Documentation and usage examples
- CI/CD pipeline with GitHub Actions

### Features
- Seven category types with customizable colors
- Single and double-line node text support
- Configurable time direction (left/right)
- Flexible year placement rules (odd/even/custom)
- Publication-ready LaTeX output
- Automatic legend generation
- Adjustable spacing, fonts, and styling
- Configuration file support (YAML/JSON)
- Configuration merging and overrides

## [0.1.0] - 2026-02-08

### Added
- Initial project structure
- Core modules: config, validator, layout_engine, latex_generator
- Comprehensive test suite
- Example files and documentation
- CLI interface
- Package configuration (pyproject.toml)
- GitHub Actions workflows
- MIT License
