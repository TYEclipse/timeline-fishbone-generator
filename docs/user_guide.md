# User Guide

This guide covers end-to-end usage of Timeline Fishbone Generator.

## Installation

See the README for installation instructions.

## CLI Usage

Run `timeline-fishbone --help` to see all options.

## Data Preparation

Prepare CSV/JSON data with the required columns:
- 年份
- 种类
- 方法名
- 引用标识

## Output

The tool generates LaTeX TikZ code that can be included in a paper or report.

## Troubleshooting

- Verify your data file format
- Check required LaTeX packages are installed
- Use `--verbose` for detailed logs
